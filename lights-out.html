Lights Out Game

Click any light to toggle it and its neighbors. Turn off all lights to win!

The Challenge:

While the tic-tac-toe game works brilliantly with radio buttons because:

  • Each cell only needs to SET a state (X or O)
  • Turn management is handled by alternating which labels are visible
  • Win detection is purely visual styling

Lights Out is fundamentally different because:

  • Each click must TOGGLE multiple cells simultaneously
  • We need labels that point to different radio buttons based on current game state
  • CSS cannot change HTML attributes like the 'for' attribute of labels

The solution would require:

  • 512 radio buttons (one for each possible game state)
  • 512 Ã— 9 labels (each state needs 9 clickable areas)
  • Each label pointing to the correct next state
  • Massive CSS to show/hide the correct labels for each state

While theoretically possible, it would be an enormous amount of HTML/CSS!
