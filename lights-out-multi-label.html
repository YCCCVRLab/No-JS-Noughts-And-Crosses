Lights Out Game

Click any light to toggle it and its neighbors. Turn off all lights to win!

The Fundamental Problem:

Even with the pattern approach, we hit the same wall:

  • HTML can only process ONE label click at a time
  • We cannot make multiple labels trigger simultaneously
  • Lights Out requires toggling multiple cells in a single action
  • This requires programming logic that HTML/CSS cannot provide

Possible solutions:

  1. Pre-calculate all states (512 radio buttons + massive HTML)
  2. Use JavaScript (breaks the HTML-only requirement)
  3. Use server-side processing (requires backend code)
  4. Accept that it's impossible with pure HTML/CSS

The tic-tac-toe game works because it only needs to SET states, not simultaneously TOGGLE multiple states. That's the key difference!
